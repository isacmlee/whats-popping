How it works

This song recommender app uses the Spotify API to gain access to your "Liked Songs" library. After obtaining the songs, a recommender model classifies 'favorite' songs based on the features of all the songs in the "Liked Songs" library. For example, if your library consists of popular songs, the recommender will recommend songs with similar popularity. There are many other song features the model could choose from (valence, instrumentalness, etc.) You can find a more in-depth explanation and the code at my GitHub
